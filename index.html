Architecture Technique

Le projet repose sur une architecture découplée : une interface web légère pour l'humain et un service "worker" (Launcher) qui travaille en tâche de fond pour calculer les créneaux disponibles.

Logique Heuristique (Python)

Algorithme "greedy" personnalisé qui calcule les créneaux libres en soustrayant les absences et les réunions existantes des disponibilités théoriques.

Intégration SQLite

Requêtes optimisées pour récupérer les contraintes (employee_availability, absences) et croiser les données en temps réel.

IA Hybride

Architecture capable de basculer entre une planification heuristique locale et un appel API vers Azure OpenAI pour les cas complexes.

Planificateur.py 
def greedy_plan(context: Dict[str,Any]) -> Dict[str,Any]:
    """
    Algorithme glouton (heuristique locale) pour planifier les tâches
    en respectant les priorités et les contraintes.
    """
    employees = {e['id']: e for e in context['employees']}
    tasks = list(context['tasks'])
    
    # Tri des tâches par priorité (Critical > High...) puis deadline
    priority_rank = {"Critical": 4, "High": 3, "Medium": 2, "Low": 1}
    tasks.sort(key=lambda t: (-priority_rank.get(t.get('priority','Medium'),2), t.get('deadline','9999-12-31')))

    plan = []
    
    for t in tasks:
        # Trouver les candidats compétents
        candidates = []
        for e in employees.values():
            if can_do_task(e, t):
                candidates.append(e)
        
        # Parcourir les jours jusqu'à la deadline
        for day in daterange(tw_from, min(tw_to, deadline)):
            if remaining[t['id']] <= 0: break
            
            for emp in candidates:
                if is_absent(absences, emp['id'], day): continue
                
                # Trouver des slots libres de 30min
                slots = list(slots_from_availability(emp['availability'], weekday_str(day)))
                
                for (s,e_) in slots:
                    if is_free(occ, emp['id'], day_str, s, e_):
                        # Assigner si respecte max heures hebdo & continu
                        plan.append({
                            'employee_id': emp['id'],
                            'task_id': t['id'],
                            'date': day_str,
                            'start_time': tstr(s),
                            'end_time': tstr(e_)
                        })
                        remaining[t['id']] -= 0.5 # -30 min

    return {"plan": plan, "notes": "Heuristique greedy appliquée."}
